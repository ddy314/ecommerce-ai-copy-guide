# 架构笔记

## 规划组件

- Web API：负责商品、评论、生成任务、导购问答等接口。
- AI Service：封装多模态大模型调用、提示词模板和情感分析流程。
- Crawler：使用 Scrapy 抓取商品评论和基础商品信息。
- PostgreSQL：保存商品、评论、生成结果、用户会话和推荐记录。
- Redis：缓存热点查询、生成结果和短期会话上下文。
- Frontend H5：面向用户的智能导购页面。
- Admin Console：面向运营或小组展示的管理后台。
